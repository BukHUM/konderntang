# Trend Today WordPress Theme

Modern news website theme for Trend Today - ทันทุกกระแสโลก

## Description

Trend Today is a clean, responsive WordPress theme designed for news and blog content with modern UI/UX. Built based on mockup designs with Tailwind CSS integration.

## Features

- Responsive design (Mobile-first)
- Modern UI/UX
- Custom post types support
- Widget areas
- Custom menu locations
- Featured images support
- Post formats support
- Translation ready
- WordPress Multisite compatible

## Requirements

- WordPress 6.0+
- PHP 8.0+
- MariaDB 10.0+

## Installation

1. Upload the `trendtoday` folder to `/wp-content/themes/`
2. Activate the theme through the 'Themes' menu in WordPress
3. Configure theme settings in Appearance > Customize
